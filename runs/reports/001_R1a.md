## Гейт 1 — ptxas

```

```
- **159 регов / 0 spill / 0 stack** — точно в диапазоне R0-прогноза (130-160r).
- Blocks/SM projection:
  - Регистры: 65536/(159×128) = 3.22 → 3 blocks/SM by registers

### Ключевые выводы NCu
- **Инверсия аксиомы FA подтверждена**: dS materialization при 8.59 GB write даёт DRAM SOL 14.6% (не 100%). Memory всё ещё изобилен для R1 chain (dS write + dS read × 2 baseline = 25.77 GB).
- **L2 Hit 96.84%** — превосходно. dS write→read через L2 работает эффективно (dS tiles temporally reused между dS-gen и потребителями dQ-new/dK-new).
- **Scheduler голод 68.79%** — memory-latency, не throughput. Соотв. ожидаемому для ядра с большим STG.
